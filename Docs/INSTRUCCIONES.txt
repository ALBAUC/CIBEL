Crear la base de datos del servicio en mysql: 
create database appwise_db;

Para crear las tablas y meter los datos, coger el archivo TFGAlinaSolonaru\Docs\backup_appwise.sql y en mysql: 
mysql -u root -p appwise_db < backup_appwise.sql

Deberian haberse creado las tablas y metido todos los datos (comprobar que en la tabla aplicacion haya 50 filas)

Abrir el servicio en eclipse y cambiar contraseña de mysql (pon la tuya) en el archivo src/main/resources/application.properties

En la carpeta AppWiseServiceSpring: mvn clean package

Ejecutar el JAR generado en target (servicio funcionando)

En Android Studio ir a la clase repository/rest/AppWiseServiceConstants y cambiar el atributo API_URL con tu ip.

Ya se podría ejecutar la aplicacion y esperar a que aparezca un mensaje diciendo que se han cargado 50 aplicaciones. (Si se ejecuta en el movil, tiene que estar conectado al mismo wifi que el ordenador)
